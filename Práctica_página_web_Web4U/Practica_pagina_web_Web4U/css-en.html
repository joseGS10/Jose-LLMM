CSS

          CSS stands for Cascading Style Sheets, and it is a language used to define the visual presentation of a web document, mostly written in HTML.

          In other words, CSS is responsible for giving style, color, layout, and formatting to a web page, separating structure (HTML) from design.

          Separation of content and presentation
          CSS allows you to separate the structure of a webpage (HTML) from its visual appearance, making design management and maintenance easier.

          Visual styles
          CSS defines how HTML elements are displayed on the screen, including colors, fonts, sizes, margins, borders, positioning, and other visual aspects.

          Cascading Style Sheets
          CSS uses a "cascading" system to apply styles, meaning styles can be defined at different levels (e.g., global, page-level, element-level) and are combined to determine the final look.

          Advantages
          CSS helps create consistent and attractive web designs, allows quick appearance changes without modifying the content, and results in cleaner, more organized code.

          Used with HTML
          CSS works alongside HTML, which defines the structure and content. HTML provides the information, and CSS gives it style.

CSS Property: color

The color property in CSS is used to define the text color of an element.

It can take values in name format (red, blue), hexadecimal (#FF0000), RGB or HSL.

Example:

          <p style="color: red;">This text is red</p>
          
Here we assign the <p> tag a style that makes the text appear in red.

CSS Property: background-color

The background-color property is used to set the background color of an HTML element.

It can be used to highlight text, sections, or other content blocks. It accepts color names, hex codes, RGB or HSL values.

Example:

          <div style="background-color: lightblue; padding: 10px;">
            Light blue background
          </div>
          
Here we apply a light blue background to the space taken by the <div> element.

CSS Property: font-size

The font-size property is used to define the size of text.

It supports absolute units (like px, pt) and relative units (like em, rem, %).

1 rem equals 1.6 px

Example:

          <p style="font-size: 24px;">
            This text is 24 pixels in size.
          </p>

CSS Property: text-align

The text-align property is used to horizontally align text within a container.

Most used values:

        left: aligns text to the left (default).
        center: centers the text.
        right: aligns text to the right.
        justify: justifies text on both sides of the container.

Example:

          <div style="text-align: center; padding: 10px;">
            <p>This paragraph is center aligned.</p>
          </div>

This paragraph is center aligned.

CSS Property: border

The border property is used to apply a border around an HTML element.

It can be defined in a single line using: border: width style color;

Most common values:

        Width: thickness of the border (e.g., 1px, 3px)
        Style:
              solid: solid line
              dashed: dashed line
              dotted: dotted line
              none: no border
        Color: can be a name (green), hex code (#00ff00), RGB, etc.

Example:

          <div style="border: 3px dashed green; padding: 10px; width: 40%;">
            <p>This block has a green dashed border, 3 pixels thick.</p>
          </div>
