HTML

          HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web.
          Define la estructura y el contenido de una página web mediante etiquetas que indican a los navegadores cómo mostrar el texto, imágenes, enlaces y otros elementos multimedia. 
          Es fundamental para la creación de cualquier sitio web. 

          Estructura
          HTML proporciona el esqueleto de la página web, organizando el contenido en diferentes secciones y jerarquías. 
          
          Contenido
          Permite incluir texto, imágenes, enlaces, videos y otros elementos multimedia. 
          
          Etiquetas
          HTML utiliza etiquetas (como <p>, <h1>, <img>, etc.) para definir estos elementos y cómo deben ser interpretados por el navegador. 
          
          No es un lenguaje de programación
          HTML no realiza cálculos ni maneja lógica, solo define la estructura y el contenido de la página. 
          
          Base de la web
          HTML es la base de casi todas las páginas web que vemos en internet, aunque a menudo se combina con CSS y JavaScript para la presentación y la interactividad. 
          
          En resumen, HTML es el lenguaje fundamental para construir la estructura y el contenido de las páginas web, permitiendo a los navegadores interpretarlas correctamente.

Etiqueta <p>

La etiqueta <p> se utiliza para definir un párrafo. Sirve para estructurar el texto en bloques separados, facilitando la lectura del contenido.

Ejemplo:

             <p>Este es un párrafo de ejemplo que demuestra el uso básico de la etiqueta p.</p>

Resultado en el navegador:

Este es un párrafo de ejemplo que demuestra el uso básico de la etiqueta p.

Etiqueta <a>

La etiqueta <a> se utiliza para crear enlaces o hipervínculos. Permite al usuario acceder a otras páginas web o a recursos como archivos, imágenes, documentos, etc.

Dentro de la etiqueta de apertura <a> colocaremos:

OBLIGATORIO. El atributo href donde se indicará la dirección de la página web o recurso a donde nos llevará cuando cliquemos sobre el enlace

OPCIONAL. El atributo target donde la opción "_black" nos permite abrir el enlace en una nueva página del navegador.

Entre la etiqueta de apertura y de cierre, se coloca el texto que le dará nombre al enlace y es lo que veremos en la WEB,

Ejemplo:

              <a href="https://www.w3schools.com" target="_blank"> W3Schools</a>

Resultado en el navegador:

Visitar W3Schools

Etiqueta <img>

La etiqueta <img> se utiliza para mostrar imágenes en una página web.

Requiere al menos el atributo src para indicar la ruta de donde se encuentra la imagen almacenada y el atributo alt para mostrar un texto alternativo en caso de que la imagen no se pueda mostrar.

También aparece en el ejemplo el atributo width que hace referencia al tamaño de la imagen

Comentar que esta eiqueta no tiene etiqueta de cierre, solo de apertura

Ejemplo:

            <img src="logos/IconoHome.png" alt="Logotipo de una casita" width="50">

Etiqueta <div>

La etiqueta <div> se usa para agrupar contenido y aplicar estilos o scripts a bloques enteros. No tiene un significado semántico específico, pero es muy útil para estructurar páginas web.

Se comporta como un contenedor donde se pueden incluir diferentes etiquetas.

Ejemplo:

          <div style="background-color: #f0f0f0; padding: 10px;">
            <p>Este párrafo está dentro de un div con fondo azul claro.</p>
            <p>Este imagen tambien está dentro del div con fondo azul claro.</p>
            <img src="logos/IconoHome.png" alt="Logotipo de una casita" width="50">
          </div>

Resultado en el navegador:

Este párrafo está dentro de un div con fondo azul claro.

Este imagen tambien está dentro del div con fondo azul claro.

Etiqueta <table>

La etiqueta <table> se utiliza para crear tablas en HTML. Dentro de una tabla se usan otras etiquetas como <tr> (fila), <th> (encabezado) y <td> (celda de datos).

Vamos a tener que usar tantas duplas <tr> </tr> como filas queramos que tenga nuestra tabla.

El contenido de la etiqueta de encabezado, <th> por defecto aparecerá en negrita y centrado dentro de la celda.

El contenido de la etiqueta de datos, <td> por defecto aparecerá alineado a la izquierda de la celda.

Ejemplo:

            <table border=1 style="width:25%;">
              <tr>
                <th>Nombre</th>
                <th>Edad</th>
              </tr>
              <tr>
                <td>Ana</td>
                <td>25</td>
              </tr>
            </table>

Resultado en el navegador:

Nombre Edad
José 51
