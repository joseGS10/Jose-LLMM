HTML

          HTML (HyperText Markup Language) is the standard language for creating web pages.
          It defines the structure and content of a web page using tags that tell browsers how to display text, images, links, and other media elements.

          Structure
          HTML provides the skeleton of the webpage, organizing content into different sections and hierarchies.

          Content
          It allows including text, images, links, videos, and other multimedia elements.

          Tags
          HTML uses tags (like <p>, <h1>, <img>, etc.) to define these elements and how they should be interpreted by the browser.

          Not a programming language
          HTML doesn't perform calculations or logic; it only defines the structure and content of the page.

          Web foundation
          HTML is the base of almost every website we see on the internet, although it is often combined with CSS and JavaScript for presentation and interactivity.

          In summary, HTML is the fundamental language for building the structure and content of web pages, enabling browsers to interpret them correctly.

<p> Tag

The <p> tag is used to define a paragraph. It helps structure text into separate blocks, making content easier to read.

Example:

<p>This is a sample paragraph demonstrating basic use of the p tag.</p>

<a> Tag

The <a> tag is used to create links or hyperlinks. It allows users to access other websites or resources like files, images, documents, etc.

REQUIRED: The href attribute specifies the destination URL of the link.

OPTIONAL: The target attribute with value "_blank" opens the link in a new browser tab.

The clickable text appears between the opening and closing tags.

Example:

<a href="https://www.w3schools.com" target="_blank"> W3Schools</a>

<img> Tag

The <img> tag is used to display images on a webpage.

It requires at least the src attribute (image source) and alt (alternative text if image cannot be displayed).

The example also uses the width attribute to control the image size.

Note: this tag is self-closing, it does not require a closing tag.

Example:

<img src="logos/IconoHome.png" alt="House logo" width="50">

<div> Tag

The <div> tag is used to group content and apply styles or scripts to entire blocks. It doesn’t have specific semantic meaning but is very useful to structure web pages.

It acts as a container for other tags.

Example:

<div style="background-color: #f0f0f0; padding: 10px;">
  <p>This paragraph is inside a light blue div.</p>
  <p>This image is also inside the div.</p>
  <img src="logos/IconoHome.png" alt="House logo" width="50">
</div>

<table> Tag

The <table> tag is used to create tables in HTML. It includes nested tags like <tr> (row), <th> (header), and <td> (data cell).

Use as many <tr> ... </tr> blocks as needed for your rows.

The <th> tag appears bold and centered by default.

The <td> tag appears left-aligned by default.

Example:

<table border=1 style="width:25%;">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>25</td>
  </tr>
</table>

Browser result:

Name Age
José 51
